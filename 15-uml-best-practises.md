# Section 15: UML Best Practises

---

## Best Practices for designing and managing UML Diagrams

- Choose the right type of UML Diagram
  - Structural Diagrams
    - Class Diagram
    - Object Diagram
    - Component Diagram
    - Composite Structure Diagram
    - Deployment Diagram
    - Package Diagram
    - Profile Diagram
  - Behavioral Diagrams
    - Use Case Diagram
    - Activity Diagram
    - State machine Diagram
    - Interaction Diagram
    - Sequence Diagram
    - Communication Diagram
    - Timing Diagram
    - Interaction Overview Diagram
- Keep UML diagrams simple
- Use UML as a communication tool
- Keep UML diagrams up-to-date
- Use UML diagrams to guide development
- Use UML diagrams for testing
- Use UML diagrams to document the system
- Validate UML diagrams
- Use UML diagrams to support reuse
- Keep UML diagrams organized
